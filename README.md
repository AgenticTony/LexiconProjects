# Lexicon Projects

This repository contains various projects completed during my Lexicon studies.

## Projects

### [Checkpoint2](./Checkpoint2/)
**Product Inventory Console Application** - A complete console application implementing product inventory management with:

- Two classes: Product and Inventory for clean separation of concerns
- LINQ operations for sorting (OrderBy) and aggregation (Sum, Where)
- Comprehensive input validation and error handling
- Interactive menu system with continuous operation capability
- Price sorting from low to high with total calculation display
- User-friendly interface with quit functionality ("q" to exit)

**Technologies:** C#, .NET 8, LINQ
**Completed:** November 2025

---

### [AssetTracking](./AssetTracking/)
**Asset Tracking System** - A comprehensive console application for tracking IT assets across multiple office locations with:

- Object-oriented design with inheritance hierarchy (Asset → Computer/Phone → specific brands)
- Abstract base classes and polymorphism for extensible asset types
- Multi-currency support with automatic price conversion (USD, EUR, SEK)
- Office location management with currency-specific pricing
- LINQ operations for sorting assets by purchase date and end-of-life calculations
- Input validation with robust error handling and user-friendly prompts
- Color-coded console output for enhanced user experience
- Automatic asset lifecycle tracking (3-year depreciation model)

**Technologies:** C#, .NET 8, OOP (Inheritance, Polymorphism, Abstraction), LINQ
**Completed:** December 2025

---

### [TodoList](./TodoList/)
**Todo List Console Application** - A feature-rich console-based todo list application built in C# demonstrating solid software engineering principles, clean architecture, and modern C# development practices with:

- Clean 4-layer architecture (UI → Services → Data → Domain) with clear separation of concerns
- Complete CRUD operations with priority system and overdue detection
- Modern C# features: nullable reference types, LINQ queries, switch expressions
- JSON file persistence with comprehensive error handling
- Professional console UI with color-coded status and formatted display
- Zero external dependencies, uses built-in .NET 8.0 libraries only
- Comprehensive test data and professional documentation
- SOLID principles implementation and thoughtful design patterns

**Technologies:** C#, .NET 8, System.Text.Json, LINQ, Clean Architecture
**Completed:** December 2025
